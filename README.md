forever - node 백그라운드 실행<br>

1. 설치<br>
$ npm install forever -g

2. 실행<br>
$ forever start node-corn.js

3. 종료<br>
리스트 조회<br>
$ forever list

리스트 숫자 입력<br>
$ forever stop 0 (리스트 숫자)
