forever - node 백그라운드 실행<br>

1. 설치
`$ npm install forever -g`

2. 실행
`$ forever start node-corn.js`

4. 실행 확인
`$ forever list`

5. 종료
`$ forever stop 0`<br>
0 = (forever list index)
