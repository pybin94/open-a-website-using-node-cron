forever - node 백그라운드 실행<br>

설치<br>
$ npm install forever -g

실행<br>
$ forever start getApi.js

종료<br>
1. 리스트 조회
$ forever list

2. 리스트 숫자 입력
$ forever stop 0 (리스트 숫자)
